# DINO 🏭

DINO based on falling sand physics, built with **C++** and **Raylib**.


## 🚀 Features
- **Simulation:** Realistic falling sand physics (Cellular Automata).
- **Interactions:** - Fire burns Wood.
  - Acid dissolves materials and contaminates water.
  - Smoke rises and accumulates on ceilings.
- **Tools:** Brush size control, Debug overlay.

## 🛠️ Build Instructions

### Prerequisites
- CMake (3.14+)
- C++ Compiler (GCC, Clang, or MSVC)

### Building
```bash
mkdir build
cd build
cmake ..
cmake --build .
