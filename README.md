# DINO - Falling Sand Simulation (C++)

A physics-based falling sand game built with C++ and Raylib. Features cellular automata logic for various elements like Sand, Water, Acid, Fire, and Smoke.

## Features
- **Elements:** Sand, Water, Wall, Wood, Fire, Smoke, Acid.
- **Interactions:** - Fire burns Wood.
  - Acid dissolves materials and contaminates water.
  - Smoke rises and accumulates on ceilings.
- **Tools:** Brush size control, Debug overlay.

## Controls
- **Left Mouse:** Draw elements.
- **Mouse Wheel:** Change brush size.
- **D:** Toggle Debug Overlay (Grid & Info).
- **R:** Reset Simulation.
- **UI Buttons:** Select elements.

## How to Build
1. Open the solution in Visual Studio 2022.
2. Ensure Raylib is installed via NuGet.
3. Build and Run (x64).
